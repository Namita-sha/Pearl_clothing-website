#PEARL

A modern luxury fashion website built with React + Vite.

---
Live Site → https://pearlwebsite-lyart.vercel.app/

Repository → https://github.com/Namita-sha/Pearl_clothing-website

---
Overview

PEARL is a responsive fashion brand frontend featuring structured category pages, curated collections, and a clean component-based architecture.

The project focuses on scalable UI design and organized routing using React.

---
#Tech Stack

React

Vite

React Router

Custom CSS

ESLint

Vercel (Deployment)
---

#Features
-Women & Men sections

-Category-based product pages

-Reusable product card components

-Wishlist & Add-to-Cart UI

-Clean routing with Vercel rewrites

-Responsive layout

---

##Installation

Clone the repository:

git clone https://github.com/Namita-sha/Pearl_clothing-website.git

cd Pearl_clothing-website

##Install dependencies:

npm install

Run locally:

npm run dev

Build for production:

npm run build
